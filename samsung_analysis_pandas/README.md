# 📊 Samsung Sales Data Analysis (Pandas Project)

## 📌 Project Overview

This project focuses on analyzing Samsung global sales data using Python (Pandas).
It covers data cleaning, transformation, and extracting meaningful business insights from raw data.

---

## 🎯 Objective

* Perform end-to-end data analysis using Pandas
* Understand sales trends and customer behavior
* Generate business insights for decision-making

---

## 🗂️ Dataset Description

The dataset includes:

* Product Name
* Category
* Country & City
* Revenue (Local & USD)
* Units Sold
* Payment Method
* Sales Channel
* Customer Segment
* Customer Age Group
* Discounts & Pricing
* Ratings and Returns

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* PyCharm

---

## 🔧 Data Cleaning & Preparation

* Handled missing values (replaced with appropriate values)
* Removed duplicates
* Standardized column names (capitalization)
* Created new columns for better analysis
* Cleaned and structured dataset for analysis

---

* Currency types

### 🔴 Level 5 — Advanced Business Insights
